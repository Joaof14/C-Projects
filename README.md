# 🚀 C Projects Collection

Repository containing **4 C language projects** developed as part of programming studies and practical applications. Each project is independent and comes with its own detailed documentation.

***

## 🗂️ Projects

1. **🏥 [Medical Clinic System](/Gerenciamento-de-Clínica)**  
   A comprehensive clinic management system with registration for doctors, patients, appointment scheduling, and reporting features.

2. **📚 [Library Management System](/Gerenciamento-de-Biblioteca)**  
   System for managing literary collections, including data loading, record updates, and file persistence.

3. **⚡ [Sorting Algorithms](/Algoritmos-de-Ordenacao)**  
   Comparative implementation of sorting algorithms (QuickSort, MergeSort, etc.) with performance analysis.

4. **🎴 [Terminal Card Game](/Jogo-de-Cartas)**  
   Multiplayer terminal game featuring card draw/discard mechanics and turn-based gameplay.

***

## 🛠️ How to Use

### Common Requirements
- GCC compiler (`sudo apt install build-essential` on Linux)
- Git for cloning the repository
- Operating system: Linux/Windows/macOS

### Basic Steps
```bash
# Clone the repository
git clone https://github.com/your-username/C-projects.git

# Navigate to a specific project
cd C-projects/Algoritmos-de-Ordenacao

# Compile (generic example)
gcc -o executable *.c

# Run
./executable
```

> **Note:** Each project has specific compilation and usage instructions in its respective README.

# 🚀 Projetos em Linguagem C

Repositório contendo **4 projetos em linguagem C** desenvolvidos como parte de estudos e aplicações práticas de programação. Cada projeto é independente e possui sua própria documentação detalhada.

---

## 🗂️ Projetos

1. **🏥 [Sistema Clínico Médico](/Gerenciamento-de-Clínica)**  
   Sistema completo para gerenciamento de clínicas com cadastro de médicos, pacientes, agendamentos e relatórios.

2. **📚 [Gerenciamento de Biblioteca](/Gerenciamento-de-Biblioteca)**  
   Sistema para controle de acervo literário com carregamento de dados, atualização de registros e persistência em arquivos.

3. **⚡ [Algoritmos de Ordenação](/Algoritmos-de-Ordenacao)**  
   Implementação comparativa de algoritmos de ordenação (QuickSort, MergeSort, etc) com análise de desempenho.

4. **🎴 [Jogo de Cartas em Terminal](/Jogo-de-Cartas)**  
   Jogo multiplayer para terminal com mecânicas de compra/descarte de cartas e turnos entre jogadores.
